About This Widget

This is a standalone, embeddable AI chatbot widget that can be integrated into any website with just two tags and one initialization call. The widget provides real-time chat functionality with streaming responses and voice call capabilities.

Key Features

  • Lightweight and fast (< 50KB gzipped)
  • Fully responsive design
  • Real-time message streaming
  • Voice call integration ready
  • Accessible with focus management
  • Customizable via CSS variables

Try It Out

Click the chat button in the bottom-right corner to start a conversation. Try asking questions or click the phone icon to test the voice call feature.
